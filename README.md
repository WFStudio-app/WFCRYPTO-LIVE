# WFCrypto Market — Real-time Dashboard

**Живые цены крипты, акций и ETF. Графики без заглушек.**

![WFCrypto Market Preview](https://via.placeholder.com/1200x600?text=WFCrypto+Market)

---

## 🚀 Что это

Дашборд для отслеживания рынков в реальном времени.  
Всё честно: реальные API, реальные свечи, никаких плавных заглушек.

| Раздел | Данные | Источник |
|--------|--------|----------|
| Crypto | Цены + свечи | Binance |
| Stocks | Цены | Finnhub |
| Stocks | Свечи | Alpha Vantage |
| ETFs | Цены | Finnhub |
| ETFs | Свечи | Alpha Vantage |

---

## ⚙️ Технологии

- HTML5 / CSS3
- JavaScript (Vanilla)
- Chart.js — графики
- API: Binance, Finnhub, Alpha Vantage

---

## 🔑 Ключи API (уже внутри)

Проект использует:

- **Finnhub** — цены акций/ETF  
- **Alpha Vantage** — исторические свечи акций/ETF  
- **Binance** — крипта (без ключа)

> В коде уже прописаны рабочие ключи. Если лимиты API будут исчерпаны — некоторые графики могут показывать ограниченные данные.

---

## 📦 Установка и запуск (‼️ВАЖНО)

1. Скачай файл `wfcrypto.html`
2. Открой в любом браузере (Chrome, Safari, Firefox)
3. Готово. Данные подтягиваются автоматически

Никаких серверов, установок или зависимостей.

---

## 🧩 Функции

- ✅ **3 вкладки** — Crypto / Stocks / ETFs
- ✅ **Реальные цены** — обновление каждые 15 секунд
- ✅ **Спарклайны** — тренд за последние часы
- ✅ **Модалка с графиком** — 4 диапазона: 1H, 4H, 1D, 1W
- ✅ **Красивый дизайн** — Google-стиль + вайб SpaceX
